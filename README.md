# 2026 FIFA World Cup Prediction Model

This project simulates the 2026 FIFA World Cup using Elo-based team ratings, roster adjustments, player production metrics, and Monte Carlo tournament simulations.

## Features
- World Cup advancement probabilities
- championship odds
- Golden Boot predictions
- roster-based rating adjustments
- team strength modeling

## Methodology
Match outcomes are simulated using Elo-based probability calculations and Monte Carlo simulations.

The model estimates:
- group winners
- advancement probabilities
- championship odds
- player scoring projections

## Data Sources
- World Football Elo Ratings
- FBref
- historical international match data
- roster availability adjustments
